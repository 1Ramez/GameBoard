Detailed Description

Represents the board for the Four-in-a-row (Connect 4) game.

This variant of Tic-Tac-Toe uses a 6x7 grid. Players drop discs into columns A player wins immediately when a four-in-a-row is formed horizontally, vertically, or diagonally
