Detailed Description

Implements the board and logic for the Word Tic-Tac-Toe game.

This variant of Tic-Tac-Toe uses a 3×3 grid. Players place letters (A–Z) instead of X/O, attempting to form valid 3-letter dictionary words. A player wins immediately when a valid word is formed horizontally, vertically, or diagonally (in either forward or reverse order).
